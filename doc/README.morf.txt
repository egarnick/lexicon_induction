The full process of producing and scoring translation hypotheses for
Morfessor-segmented text requires the following 6 steps.  Each step has a
corresponding README file in

/g/ssli/data/lorelei/Software/Morfessor/morf/doc/detail

Steps and README files
1.  Prepare the seed lexicon
  README.morf.lex.txt
2.  Create word counts
  README.morf.uzb_counts.txt
3.  Create word vectors
  README.morf.uzb_vectors.txt
4.  Prepare the test items
  README.extract_test.tst
5.  Compare Uzbek and English word vectors to produce translation hypotheses
  README.morf.compare_vectors.txt
6.  Score translations
  README.score_trans.txt
